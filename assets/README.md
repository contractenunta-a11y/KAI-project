# KAI Branding Pack

This pack contains official KAI logo exports in multiple formats and sizes.

## Specs
- Mark: `< | !`
- Primary color: `#8AA0E0`
- Background: black (main version)

## Folders
- `logo/` — master PNG + embedded SVG wrappers (note: embedded SVG = raster inside SVG)
- `icons/png/` — square token icons (on-black + transparent variants)
- `social/` — X profile/banner + OpenGraph images
- `web/` — favicon.ico + apple touch icons
- `mobile/` — android icons
- `master/` — master exports

## Recommended for BaseScan / wallets / listings
- `icons/png/256.png`
- `icons/png/512.png`
- `master/kai_logo_1024.png`

## Transparent exports
Files ending with `_transparent.png` have the black background removed (best effort).
