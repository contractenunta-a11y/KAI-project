# Contributing to KAI

Thank you for your interest in contributing to the KAI project.

## How to Contribute
1. Fork this repository.
2. Create a new branch for your changes.
3. Follow clear commit messages (see below).
4. Submit a Pull Request with a detailed explanation.

## Commit Message Guidelines
- Use short, descriptive messages.
- Example: "Add asset icons", "Update documentation", "Fix contract reference".

## Code of Conduct
Be respectful, constructive, and maintain professionalism at all times.
